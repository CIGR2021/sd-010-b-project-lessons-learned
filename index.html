Os comandos que você utilizará com mais frequência são: git status (para verificar o que está em vermelho - fora do stage - e o que está em verde - no stage) git add (para adicionar arquivos ao stage do Git) git commit (para criar um commit com os arquivos que estão no stage do Git) git push -u nome-da-branch (para enviar o commit para o repositório remoto na primeira vez que fizer o push de uma nova branch) git push (para enviar o commit para o repositório remoto após o passo anterior)
