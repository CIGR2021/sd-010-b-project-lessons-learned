Meu aprendizado na Trybe

A cada dia que passa meu aprendizado na trybe vem crescendo de forma significativa. Meu conhecimento em front-end era quase zero e após poucas semanas eu estou conseguindo criar páginas usando HTML e CSS, fazendo o versionamento do código com o Git e GitHub e utilizando o Unix.

Além das hard-skills a trybe trabalha com com Soft-skills dos estudantes, onde já pude me atentar para vários pontos de melhoria que preciso exercitar para poder me ser bem avaliado em um processo seletivo, ou mesmo dentro da empresa.

Uma passsagem na trybe

Uma passagem que me marcou foi na construção do portifólio, quando no início era apenas uma página com textos HTML e quando foi introduzido o estilo com CSS a página ganhou uma "cara" nova.
