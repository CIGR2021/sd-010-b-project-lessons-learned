Na Trybe aprendemos sobre Desenvolvimento Web Fullstack, o que inclui ferramentas front-end e back-end, bem como uso de ferramentas de trabalho como Zoom, Slack, Git e Gihub. E o uso de Sistemas Operacionais Linux, a usar Bash e Node-js. Em front-end aprendemos HTML e CSS. E para back-end, JavaScript e SQL. Aprenderemos tanto sobre bando de dados relacionais quanto os não-relacionais, como MongoDB.

Um detalhe interessante no curso é o foco no mercado de trabalho, o curso é todo estruturado na rotina de quem já trabalha como desenvolvedor remoto, seus exercícios são baseados em situações reais.
